## How clone this repository?

💻 In CLion

>Open the cloning tool: Go to the main menu and select Git | Clone. If you don't see the Git option, look for VCS | Get from Version Control.

>Sign in to GitHub: In the window that appears, choose GitHub on the left panel and sign in with your account. You can use a token or click Log In via GitHub to authorize access in your browser.

>Select the repository: Choose the repository you want to clone from the list of your projects on GitHub.

>Set the location: In the Directory field, specify the path on your computer where the project will be saved.

>Finish: Click Clone. CLion will ask if you want to create a project based on these files; confirm with Yes.

🖥️ In Visual Studio

>Open the cloning tool: Open Visual Studio and, on the top menu, select Git | Clone Repository.

>Note: If this is your first time using Git, the option might simply be called Clone. If the Git menu doesn't appear, go to Tools > Options, expand Source Control, and make sure the Plug-in Selection is set to Git.

>Sign in to GitHub: In the Clone a repository window, under the Browse a repository section, select GitHub. If you haven't linked your account, click Sign in and authorize Visual Studio.

>Select the repository: After signing in, you'll see a list of your repositories. Select the one you want to clone.

>Set the location: In the Repository location field, paste the URL of the repository (which you can copy from GitHub). In the Path field, define the folder where it will be saved.

>Finish: Click Clone. Visual Studio will download the repository and show you the list of solutions (projects) available to open.

📝 In Visual Studio Code

>Open the command palette: Press Ctrl+Shift+P (Windows/Linux) or Cmd+Shift+P (Mac) to open it.

>Search for the clone command: Type Git: Clone and select that option.

>Provide the URL: You will be prompted to enter the URL of the GitHub repository you want to clone. Copy it from the repository's page on GitHub (the green "Code" button).

>Select the destination folder: Choose the folder on your computer where you want to save the project.

>Open the repository: Once cloned, VS Code will offer to open the cloned repository.

>Alternatively, you can use the Source Control panel in the sidebar (the Git icon) and click the Clone Repository button to follow the same steps.
